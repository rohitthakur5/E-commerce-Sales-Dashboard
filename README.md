# 📊 E-commerce Sales Performance Dashboard

## 📝 Project Overview
This interactive dashboard was developed to transform raw sales data into actionable business insights. It provides a comprehensive view of sales trends, customer behavior, and key performance indicators (KPIs) to support data-driven decision-making.

## 🚀 Key Insights
* **Monthly Sales Trends**: A comparative analysis of total revenue and order volume throughout the year.
* **Customer Demographics**: Detailed breakdown of sales distribution by gender (64% Women vs 36% Men) and various age groups.
* **Top Revenue Zones**: Identification of the top 5 performing states (Maharashtra, Karnataka, Uttar Pradesh, etc.) contributing to the highest sales.
* **Channel Performance**: Revenue contribution across major e-commerce platforms like Amazon, Flipkart, Myntra, and Ajio.
* **Order Status**: Tracking operational efficiency through delivery success rates (92% Delivered) and refund/cancellation metrics.

## 🛠️ Technical Skills Used
* **Data Cleaning & Transformation**: Processing raw datasets for accurate visualization.
* **Advanced Pivot Tables**: Summarizing complex data points.
* **Interactive Visualization**: Implementation of Slicers and Timelines for dynamic filtering.
* **Dashboard Design**: Creating a professional, user-friendly layout with a focused aesthetic.
